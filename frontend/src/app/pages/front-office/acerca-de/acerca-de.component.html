Importante, esta página no pertenece a una tienda real y sólo es una aplicación de ejemplo.

El front-end de ésta aplicación fue desarrollado con el framework Angular en su versión 11 y lenguaje typescript

Angular es un framework opensource desarrollado por Google para facilitar la creación y programación de aplicaciones web de una sola página

Además se ha utilizado la tecnología de bootstrap para todo el frontend

Es así que el frontend de la aplicación, para su front-office, o tienda, se utilizó el template Essence mientras que para el back-office o administración de la tienda se utilizó el template Adminkit

Además, para los end-points del backend la aplicación utiliza el framework laravel en su versión 8.

El objetivo de Laravel es el de ser un framework que permita el uso de una sintaxis refinada y expresiva para crear código de forma sencilla, evitando el «código espagueti» y permitiendo multitud de funcionalidades

Simular pagos con webpay:

Para simular pagos con webpay, al seleccionar el botón Pagar con Paypal, ubicado en la pantalla de Datos de envío, la aplicación redireccionará a WebPay, allí los pasos a seguir son los siguientes:

1- Primero se debe seleccionar la opción debito.
2- A la derecha de la pantalla seleccionar como banco TEST COMMERCE BANK.
3- Como número de tarjeta se puede utilizar el número 4051 8842 3993 7763 y seleccionar Pagar $xx.xxx (*).
4- En la pantalla que aparecerá, ingresar como usuario 11.111.111-1 y como contraseãa 123.
5- En la siguiente pantalla seleccionar Aceptar para simular un pago exitoso o Cancelar para simular un pago no autorizado.

Finalmente Webpay, redireccionará de regreso a la tienda, donde se mostrará una pantalla con el resumen del Pago.
* Para probar otras tarjetas y medios de pago con Webpay, los puedes obtener en la página de Transbank developers en la sección Tarjetas de Prueba
